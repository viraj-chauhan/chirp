# NagarVaad – Setup Instructions

## Prerequisites

Install **Node.js v18+** from: https://nodejs.org/en/download

After installation, verify:
```
node --version   # should show v18+ or v20+
npm --version    # should show 9+ or 10+
```

## Running the Project

```bash
cd civic-portal
npm install
npm run dev
```

Open: http://localhost:3000

## Architecture Notes

- **No backend/database** – all data is in `lib/data.ts`
- **Auth** – localStorage-based (email/password comparison against hardcoded users)
- **AI features** – simulated (pattern matching in education; pre-written summaries for debates)
- **Real-time debates** – simulated with pre-populated messages

## Production Roadmap

To make this production-ready, add:
1. **Database**: PostgreSQL + Prisma (or Supabase)
2. **Auth**: NextAuth.js or Clerk
3. **Real-time**: Socket.io or Supabase Realtime for live debates
4. **AI**: OpenAI/Claude API for actual transcript generation
5. **Backend**: Next.js API routes for CRUD operations
6. **Data sources**: MPLADS API, ECI affidavit scraper for real government data
